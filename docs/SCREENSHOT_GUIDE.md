# Captures pour publication

Captures recommandees pour la fiche publique :

1. Lecteur YouTube normal avec le badge `Fin HH:MM` visible.
2. Changement de vitesse de lecture ouvert, pour montrer que le calcul s'adapte.
3. Mode theatre ou plein ecran avec le badge integre aux controles.

Conseils :

- Utiliser une video courte et non sensible.
- Masquer les informations personnelles du navigateur.
- Garder le badge lisible sans zoom excessif.
- Exporter en PNG ou JPG selon les contraintes du store.
